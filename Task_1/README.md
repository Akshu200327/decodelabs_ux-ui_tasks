# Fitness Application – UX Research Case Study

## Overview

This project was created as part of my UI/UX learning journey. The goal was to understand the challenges faced by busy working professionals who want to stay fit but struggle with consistency due to work pressure and time management issues.

Instead of directly designing screens, I started by understanding the user's problems, goals, motivations, and daily habits. This helped me identify the real challenges behind fitness consistency.

---

## Problem Statement

Many working professionals want to maintain a healthy lifestyle but often miss workouts and meals because of busy schedules, mental fatigue, and poor routine management.

The challenge was to understand these users and define a solution that could help them build consistent fitness habits.

---

## Research Process

For this project, I created:

* Target User Profile
* User Persona
* User Journey Map
* Feature Prioritization
* Information Architecture
* User Flow
* Screen List
* Empathy Map

These activities helped me understand user behavior before moving into the design phase.

---

## Key Findings

* Users know fitness is important but struggle with execution.
* Lack of routine is a bigger problem than lack of knowledge.
* Simple reminders and structured daily plans can improve consistency.
* Progress tracking can motivate users to continue their fitness journey.

---

## What I Learned

Through this project, I learned how UX research helps uncover real user problems before designing solutions. I also gained experience in creating personas, empathy maps, user flows, and information architecture.

This project strengthened my understanding of user-centered design and the importance of solving problems before focusing on visuals.
