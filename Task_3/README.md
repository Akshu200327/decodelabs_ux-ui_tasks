# Farmly - Organic Food Delivery UI

Farmly is a mobile app UI design concept created for a premium organic food delivery service.

The objective of this project was to transform low-fidelity wireframes into a complete high-fidelity mobile interface while focusing on visual hierarchy, branding, typography, spacing, and user experience.

## Project Goal

To create a fresh and modern user interface that communicates:

- Trust
- Health
- Freshness
- Simplicity

The design focuses on helping users browse organic products, view product details, and add items to their cart with ease.

## Screens Designed

### Splash Screen
Introduces the Farmly brand with a clean and minimal first impression.

### Product Listing Screen
Allows users to browse categories, search products, and quickly add items to their cart.

### Product Detail Screen
Provides detailed information about a product including pricing, description, nutritional information, and purchase actions.
