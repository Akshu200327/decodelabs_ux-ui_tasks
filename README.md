# Decodelabs UI/UX Internship Tasks

This repository contains the UI/UX design projects completed during my internship learning journey.

The projects focus on different stages of the design process including user research, wireframing, visual design, and high-fidelity interface creation.

## Projects Included

### Task 1 – Fitness App UX Research

A research-based project focused on understanding the challenges faced by working professionals in maintaining fitness consistency.

Key Activities:

* User Research
* User Persona Creation
* Problem Identification
* UX Analysis

### Task 2 – E-Commerce Wireframe Design

A low-fidelity wireframing project for an e-commerce platform.

Key Activities:

* User Flow Planning
* Layout Structuring
* Homepage Wireframe Creation
* Information Architecture

### Task 3 – Farmly Organic Food Delivery App

A high-fidelity UI design project for an organic grocery delivery application.

Key Activities:

* Style Guide Creation
* Visual Design System
* Splash Screen Design
* Product Listing Screen
* Product Detail Screen
